Credit Card Analysis

Canadian Credit Card Analyzer

Key Features:

  • Automated Web Scraping
  • Reward Rate Analysis
  • Personalized Value Calculator
  • Card Ranking System

Sample Code:


# Initialize scraper and get data
scraper = CreditCardScraper()
scraper.scrape_ratehub()

# Initialize analyzer with your spending
analyzer = CreditCardAnalyzer(scraper.cards_data)
spending_profile = {
    'groceries': 6000,  # $6000/year
    'gas': 2400,        # $2400/year
    'dining': 3600,     # $3600/year
    'travel': 2000      # $2000/year
}

# Rank cards based on your preferences
ranked_cards = analyzer.rank_cards(
    spending_profile, 
    {'welcome_bonus': 0.3, 'interest_rate': -0.2}
)
